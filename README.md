## Custom Firmware

- [ARK-4](https://github.com/PSP-Archive/ARK-4/) – Main custom firmware. Supports homebrew, plugins, recovery...
- [FasterARK](https://github.com/PSP-Arkfive/FasterARK/) - ARK-5 loader/installer

## Emulators

- [GameBrew – PSP Emulators](https://www.gamebrew.org/wiki/List_of_PSP_homebrew_emulators) - Full list of emulators.
- [PSP Archive – All Emulators](https://psp-archive.github.io/emulators/all-emulators-for-PSP.html) - Emulator list with compatibility notes.
- [TempGBA4PSP](https://github.com/andymcca/TempGBA4PSP-mod) - GBA emulator.
- [gpSP Kai](https://www.gamebrew.org/wiki/UO_g%D1%80SP_Kai_PSP) - GBA emulator.
- [Snes9X Euphoria](https://www.gamebrew.org/wiki/Snes9X_Euphoria_PSP) - SNES emulator.
- [DaedalusX64](https://github.com/DaedalusX64/daedalus) - N64 emulator.
- [Cult-of-GBA/BIOS](https://github.com/Cult-of-GBA/BIOS) - GBA open-source replacement BIOS.

⚠️ **GBA emulators**: gpSP Kai and TempGBA4PSP need a `gba_bios.bin` file. Dump yours or download one from the internet and put it at the root of the emulator directory.

## Plugins

- [CXMB](https://github.com/PSP-Archive/CXMB) - Loads custom `.ctf` themes. Needed for any custom theme on ARK-4.
- [CWCheat Database](https://github.com/Saramagrean/CWCheat-Database-Plus-) - Big cheat database for CWCheat/ARK-4 and PPSSPP.
- [TempAR](https://github.com/raing3/tempar) - Cheat plugin supporting both CWCheat and Action Replay codes.
- [Recommended Plugins (ARK-4 Wiki)](https://github.com/PSP-Archive/ARK-4/wiki/Recommended-Plugins) - Official plugins picks.

## Themes

- [PTF and CTF themes - pspunk](https://www.pspunk.com/psp-themes/) – A collection of PTF (work on official firmware) and CTF themes
- [PSP-Themes Collection](https://github.com/Shenron0/PSP-Themes) - Collection of CTF themes for PSP.
- [Themestation](https://psp.org.tr/themes) – Largest PSP theme archive.

⚠️ [CXMB](https://github.com/PSP-Archive/CXMB) is required to load `.ctf` themes. See plugins above.

## Homebrew Games

- [GameBrew – PSP Homebrew Games](https://www.gamebrew.org/wiki/List_of_PSP_homebrew_games) - Full list of homebrew games.
- [CSPSP](https://www.gamebrew.org/wiki/CSPSP) - Top-down multiplayer shooter, Counter-Strike style.
- [Kurok](https://www.gamebrew.org/wiki/Kurok_PSP) - FPS inspired by GoldenEye/Turok.
- [Lamecraft](https://www.gamebrew.org/wiki/Lamecraft_PSP) - Minecraft-style sandbox game.
- [Minecraft PSP 2.0](https://www.moddb.com/mods/minecraft-psp/downloads/minecraft-psp-20-release) - Minecraft-style sandbox game, based on Lamecraft.
- [nzportable (NZ:Portable)](https://github.com/nzp-team/nzportable) - Call of Duty: Zombies demake.
- [Dino Run PSP](https://www.gamebrew.org/wiki/Dino_Run_PSP) – Port of chrome's dinosaur game.
- [Paper PSP](https://github.com/jeromeberg/paper-psp) – Paper.io clone for PSP.

## Tools

- [PPSSPP](https://www.ppsspp.org/) – PSP emulator for Android, Windows, Mac, Linux and iOS.

### ISO

- [ciso](https://github.com/jamie/ciso) - Simple command line utility to compress PSP iso files.
- [cisoplus](https://github.com/meoow/cisoplus) - CSO compressor with extra options.
- [maxcso](https://github.com/unknownbrackets/maxcso) - Fastest CSO compressor, smallest file size.

## Development

- [pspdev](https://github.com/pspdev/pspdev) – Full PSP dev environment (start here).
- [PSPDEV Docs](https://pspdev.github.io/) – Official documentation for PSPDEV.
- [pspsdk](https://github.com/pspdev/pspsdk) – Open-source SDK for PSP homebrew development.
- [rust-psp](https://github.com/overdrivenpotato/rust-psp) - Rust on PSP (no PSPSDK dependency).
- [OSLib MODv2](https://github.com/PSP-Archive/oslibmodv2) - Simple 2D graphics library, good for beginners.
- [GU Library Reference](https://pspdev.github.io/pspsdk/group__GU.html) - Docs for libGU, PSP's hardware graphics API.
- [Lua Player HM](https://github.com/PSP-Archive/Lua-Player-HM) - LUA interpreter for PSP.
- [PSP Homebrew Development](https://github.com/jdriselvato/PSP-Homebrew-Development) - Examples to learn homebrew development.
- [PSP Programming](https://github.com/robfernan/PSP-Programming) - C tutorials using PSPSDK and SDL2.
- [PSP Development — Wikibooks](https://en.wikibooks.org/wiki/PSP_Development) - Wiki on PSP development.
- [PSP Programming Languages Overview](https://psp-archive.github.io/apps/PSP_programming_languages.html) - Overview of the language options available to PSP programmers.
- [PSPLINK](https://github.com/pspdev/psplinkusb/) – Start and debug homebrew through USB.
